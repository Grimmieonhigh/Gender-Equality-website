This page explains the design, structure, and development of my assigned pages. My aim was to create interactive, visually consistent, and meaningful pages that both inform users and provide a smooth user experience.

📄 User Profile

The User Profile page was designed to be interactive and engaging. I used JavaScript to guide users through a step-by-step process where they enter their details, views, and personal goals related to gender equality.

A progress bar was included to visually show completion, improving user experience. I also implemented input validation so that the age field only accepts numbers, while name and role only accept letters. This ensures accurate data and improves usability.

The design uses a clean card layout and a brown accent colour scheme to maintain consistency with my other pages while still fitting into the overall website design.

🗺️ Sitemap

The sitemap was created using SVG to visually represent the structure of the website. This allowed me to precisely position elements and ensure the layout is clear and readable.

I organised the sitemap into three sections: entry page, main pages, and content pages. The layout follows the same order as the navigation bar, making it easier for users to understand.

I carefully adjusted spacing and alignment so that all elements sit evenly between connector lines. This improved the visual balance and professionalism of the design.

View Sitemap

📊 Content Page - Gender Pay Gap

My content page focuses on the gender pay gap, a significant issue affecting women globally. I aimed to make the content both informative and personal, explaining not only the facts but also why this issue matters.

I used structured sections, including key facts, causes, impacts, and solutions, to guide the reader through the topic clearly. Images were added throughout the page to improve engagement and break up large sections of text.

I also used a more human and passionate tone to make the content more relatable, highlighting how the gender pay gap affects women's opportunities, confidence, and long-term financial security.

View Content Page

⚠️ Challenges and Solutions

One challenge I faced was ensuring my styles did not affect other group members' pages. I solved this by using page-specific classes to scope my CSS.

Another challenge was aligning elements in the sitemap correctly. This required careful adjustment of positions and spacing within the SVG.

♿ Accessibility

I used clear headings, readable text, and alt text for images to improve accessibility. The structured layout and navigation also help users easily move through the site.
